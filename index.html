VRChat Cloud Gallery
使い方ガイド

ワールドをアップロードし直さなくても、
写真や画像をVRChatの中で表示・更新できるツールです。

What's this?
このツールでできること

VRChatのワールドに「ギャラリーウォール」を置いておけば、
あとはパソコンで画像を選んでボタンを押すだけで展示内容が更新されます。
ワールドを作り直したり、Unityを起動したりする必要はありません。

Page 2 — Getting Started

はじめてのセットアップ

Getting Started
はじめてのセットアップ

最初の1回だけ準備が必要です。順番にやってみてください。
慣れればおよそ20〜30分で完了します。

1

ライセンスキーを入力する

購入完了メールに記載されているライセンスキー（例：VGM-LITE-XXXXXXXXXX）を アプリ起動時の画面に入力してください。このPCに保存されます。

2

GitHubアカウントを作る（無料）

https://github.com/signup からアカウントを作成してください。メールアドレスがあれば無料で使えます。 すでに持っている方はスキップしてください。

3

GitHubのアクセストークンを取得する

GitHubにログインして、 アクセストークンの発行ページ を開きます。「Generate token」ボタンを押してコピーしておいてください。 （この画面を閉じると二度と表示されないので注意）

4

管理ツールに設定を入力する

アプリの「設定」画面を開き、取得したアクセストークンとリポジトリ名 （例：あなたのID/my-gallery）を入力して保存してください。 入力した情報はこのPC内に暗号化して保存されます。

5

Unityでワールドにギャラリーを配置する

同梱の VRChatGallery-Base.unitypackage をUnityプロジェクトにインポートし、 ワールドに「GalleryPanel」オブジェクトを置きます。 Inspector で galleryId を設定してください（例：wall_a）。 ワールドをアップロードするのはこの1回だけです。

Add Images
画像を追加する
A

ドラッグ＆ドロップで追加

エクスプローラーから画像を「画像を追加」エリアへ直接ドロップします。複数ファイルも一度にドロップできます。JPG・PNG・WebP・BMPに対応しています。

B

「画像を追加」エリアをクリックするとファイル選択ダイアログが開きます。Ctrlキーで複数選択できます。

C

左メニューから「Google Drive取込」を選び、ファイルまたはフォルダのURLを貼り付けます。フォルダを指定すると中の画像をすべて一括取り込みできます。

✅
追加した画像は自動的にリサイズ・圧縮・サムネイル生成が行われます。重い画像でもそのまま入れて大丈夫です。
Manage Images
画像を並び替え・削除する
⠿

ドラッグで並び替える

各画像カードの左端にある ⠿ マークをドラッグすると順番を変えられます。スライドショーはこの順番で再生されます。

✕

ホバーして削除ボタンを押す

カードにマウスを乗せると右上に「✕」ボタンが現れます。確認ダイアログの後に削除されます。ローカルの元ファイルは消えません。

Aa

キャプションを付ける（任意）

カード下部のテキスト部分をクリックすると説明文を入力できます。VRChatで画像を拡大したときに表示されます。

GitHub Upload
GitHubにアップロードする
1

上部の「アップロード」タブを開く

「GitHub Pages Upload」パネルが表示されます。

2

「認証確認」ボタンを押す

「認証成功: @あなたのID」と表示されれば準備OKです。

3

「アップロード開始」を押す

画像ファイルとギャラリー設定ファイルが順番にアップロードされます。枚数によって数十秒〜数分かかります。

4

完了後、初回だけURLをUnityに設定する

「GitHub Pages URL: https://○○.github.io/○○/」が表示されます。このURLをUnityのGalleryLoaderの「Manifest Url」に設定してワールドを再アップロードしてください。2回目以降は不要です。

R2 Upload — Standard / Pro
Cloudflare R2にアップロードする
ℹ️
R2を使うには事前にCloudflareアカウントの作成が必要です。
1

CloudflareでR2バケットを作る

Cloudflareにログイン → 左メニューの「R2」→「バケットを作成」。好きな名前でOKです（例：my-gallery）。作成後「設定」タブでパブリックアクセスを有効にします。

2

APIトークンを発行する

R2バケットの「APIトークンを管理」からトークンを作成します。権限は「オブジェクトの読み取りと書き込み」。表示される「アクセスキーID」と「シークレットアクセスキー」をコピーします。

⚠️
シークレットアクセスキーはこの画面を閉じると再表示できません。必ずコピーしてください。
3

管理ツールの設定画面でR2情報を入力する

「設定」→「Cloudflare R2」に Account ID・Bucket名・Access Key ID・Secretを入力します。「接続確認」ボタンで動作テストができます。

4

「アップロード開始」を押す

R2はGitHubより大幅に速く、50枚程度なら数秒で完了します。完了後にbaseURLをUnityのManifest Urlにコピーしてワールドをアップロードしてください（初回のみ）。

Update Flow
2回目以降の更新手順

セットアップ完了後の毎回の手順はこれだけです。

1

管理ツールで画像を追加・並び替えする

ドラッグ＆ドロップで追加、⠿ハンドルで並び替え。

2

「アップロード開始」を押す

GitHub または R2 に変更分が反映されます。

3

VRChatでワールドに入り直す

以上で完了です。ワールドの再アップロードは不要です。

Page 5 — FAQ & Support

よくある質問・サポート

困ったときはまずここを確認してください。

Troubleshooting
うまくいかないときのチェックリスト

🥽 VRChatで画像が表示されない

  • 管理ツールでアップロードが完了しているか確認する
  • GalleryLoader の Manifest Url が正しいURLになっているか確認する
  • 管理ツールの Gallery Id と Unity の Inspector の Gallery Id が完全一致しているか確認する
  • ワールドに入り直す（キャッシュが残っている場合がある）

📤 アップロードがエラーになる

  • インターネットに接続しているか確認する
  • 設定画面で「認証確認」を押して接続テストをする
  • GitHubトークンの有効期限が切れていないか確認する
  • リポジトリ名が ID/リポジトリ名 の形式になっているか確認する

🔑 ライセンスキーが認証されない

  • キーをコピー＆ペーストして入力する（手入力はミスしやすい）
  • VGM- から末尾まで全体がコピーされているか確認する
  • それでも通らない場合はBOOTHから販売者に連絡する

📁 Google Driveから取り込めない

  • 共有設定が「リンクを知っている全員が閲覧可」になっているか確認する
  • URLが drive.google.com/file/d/ または /drive/folders/ の形式か確認する
FAQ
よくある質問
無料で使えますか？
このツール自体は有料（ライセンスキーが必要）ですが、 画像を保存するクラウドサービスは無料枠があります。 GitHubは個人利用なら完全無料。Cloudflare R2も月10GBまで無料です。 一般的な使い方では費用はかかりません。
アップロードした画像は誰でも見られますか？
GitHubを使う場合は画像が公開状態になります。 「URLを知らないと見られない」という状況にはなりますが、完全な非公開ではありません。 Cloudflare R2（Standard以上）でも同様です。 公開したくない画像のご利用はご注意ください。
Unityの知識がなくても使えますか？
初回セットアップ時にUnityでオブジェクトを配置する手順があります（一度だけ）。 セットアップ完了後は管理ツールだけで操作でき、Unityは不要です。 UnityとVRChat SDKの基本的な使い方を知っていれば問題なく設定できます。
Quest（スタンドアロンVR）でも動きますか？
はい、Quest対応しています。 メモリを節約するために画像は小さいサムネイルから先に読み込み、 クリックしたときだけ大きい画像を取得する仕組みになっています。 画像枚数の目安は1ギャラリーにつき64枚以内を推奨しています。
画像はどのくらいの解像度が適切ですか？
管理ツールが自動でリサイズします。 原則として元の画像をそのまま入れれば問題ありません。 参考として、VRChatの最大テクスチャサイズに合わせて フルサイズは2048px以内、サムネイルは512px以内に自動縮小されます。
LiteからStandardにアップグレードするとどうなりますか？
BOOTHの差額商品を購入すると新しいライセンスキーが届きます。 管理ツールの設定画面でキーを入力し直すだけでStandard機能が使えるようになります。 ワールドの再アップロードは不要で、これまでの設定もそのまま使えます。
PCを変えたときはどうすればいいですか？
新しいPCに管理ツールをインストールして、同じライセンスキーを入力してください。 GitHubやR2のアカウント情報も再入力が必要ですが、 クラウドに保存された画像や設定はそのまま使えます。
