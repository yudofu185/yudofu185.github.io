Getting Started
はじめてのセットアップ

最初の1回だけ準備が必要です。順番にやってみてください。
慣れればおよそ20〜30分で完了します。

1

ライセンスキーを入力する

購入完了メールに記載されているライセンスキー（例：VGM-LITE-XXXXXXXXXX）を アプリ起動時の画面に入力してください。このPCに保存されます。

2

GitHubアカウントを作る（無料）

https://github.com/signup からアカウントを作成してください。メールアドレスがあれば無料で使えます。 すでに持っている方はスキップしてください。

3

GitHubのアクセストークンを取得する

GitHubにログインして、 アクセストークンの発行ページ を開きます。「Generate token」ボタンを押してコピーしておいてください。 （この画面を閉じると二度と表示されないので注意）

4

管理ツールに設定を入力する

アプリの「設定」画面を開き、取得したアクセストークンとリポジトリ名 （例：あなたのID/my-gallery）を入力して保存してください。 入力した情報はこのPC内に暗号化して保存されます。

5

Unityでワールドにギャラリーを配置する

同梱の VRChatGallery-Base.unitypackage をUnityプロジェクトにインポートし、 ワールドに「GalleryPanel」オブジェクトを置きます。 Inspector で galleryId を設定してください（例：wall_a）。 ワールドをアップロードするのはこの1回だけです。

Daily Use
画像を更新するときの手順

セットアップが終わった後は、毎回この3ステップだけです。

1

画像を追加する

管理ツールを開き、表示したい画像を画面にドラッグ＆ドロップします。 JPG・PNG・WebPなどの画像ファイルに対応しています。 リサイズや圧縮は自動で行われます。

2

並び順を整える（任意）

追加した画像は自由に並び替えられます。 表示モード（スライドショー・グリッドなど）もここで選べます。

3

「アップロード」ボタンを押す

アップロードが完了すると、VRChat内のギャラリーが自動的に更新されます。 ワールドのアップロードは不要です。

✅
ワールドに入り直すだけで最新の画像が表示されます。 VRChat内にいる最中でも、次回ワールドを起動したときに自動で反映されます。
Google Drive Import
Google Driveから直接取り込む

Google Driveに保存している画像を、URLを貼り付けるだけでそのまま取り込めます。 スマートフォンで撮影してDriveに保存→PCで開く、という使い方にも便利です。

⚠️
取り込むファイルまたはフォルダは Google Drive の共有設定で 「リンクを知っている全員が閲覧可」に設定されている必要があります。 「自分のみ」の設定ではダウンロードできません。
1

Google DriveでファイルのURLをコピーする

取り込みたい画像（またはフォルダ）を右クリック →「リンクをコピー」

2

管理ツールの「Google Drive取込」欄に貼り付ける

URLを貼り付けると、ファイルかフォルダかが自動で判別されます。

3

「インポート」ボタンを押す

ダウンロードと画像処理が自動で行われます。フォルダ内の画像は一括取込できます。

FAQ
よくある質問
無料で使えますか？
このツール自体は有料（ライセンスキーが必要）ですが、 画像を保存するクラウドサービスは無料枠があります。 GitHubは個人利用なら完全無料。Cloudflare R2も月10GBまで無料です。 一般的な使い方では費用はかかりません。
アップロードした画像は誰でも見られますか？
GitHubを使う場合は画像が公開状態になります。 「URLを知らないと見られない」という状況にはなりますが、完全な非公開ではありません。 Cloudflare R2（Standard以上）でも同様です。 公開したくない画像のご利用はご注意ください。
Unityの知識がなくても使えますか？
初回セットアップ時にUnityでオブジェクトを配置する手順があります（一度だけ）。 セットアップ完了後は管理ツールだけで操作でき、Unityは不要です。 UnityとVRChat SDKの基本的な使い方を知っていれば問題なく設定できます。
Quest（スタンドアロンVR）でも動きますか？
はい、Quest対応しています。 メモリを節約するために画像は小さいサムネイルから先に読み込み、 クリックしたときだけ大きい画像を取得する仕組みになっています。 画像枚数の目安は1ギャラリーにつき64枚以内を推奨しています。
画像はどのくらいの解像度が適切ですか？
管理ツールが自動でリサイズします。 原則として元の画像をそのまま入れれば問題ありません。 参考として、VRChatの最大テクスチャサイズに合わせて フルサイズは2048px以内、サムネイルは512px以内に自動縮小されます。
LiteからStandardにアップグレードするとどうなりますか？
BOOTHの差額商品を購入すると新しいライセンスキーが届きます。 管理ツールの設定画面でキーを入力し直すだけでStandard機能が使えるようになります。 ワールドの再アップロードは不要で、これまでの設定もそのまま使えます。
PCを変えたときはどうすればいいですか？
新しいPCに管理ツールをインストールして、同じライセンスキーを入力してください。 GitHubやR2のアカウント情報も再入力が必要ですが、 クラウドに保存された画像や設定はそのまま使えます。
Glossary

GitHub
ファイルをインターネット上に保存・公開できる無料サービス。このツールでは画像の保存・配信に使います。
GitHub Pages
GitHubのリポジトリ内容をウェブサイトとして公開する機能。画像の配信に使います（無料）。
Cloudflare R2
Cloudflareが提供するファイルストレージ。GitHubより読み込みが速く、大量の画像に向いています（月10GBまで無料）。
アクセストークン

リポジトリ

UdonSharp

unitypackage
Unityに追加する機能をまとめたファイル。Unityのメニューからダブルクリックするだけで取り込めます。
galleryId
各ギャラリーパネルに付ける名前。管理ツール側とUnity側で同じ文字列を設定することで対応させます（例：wall_a）。
